## PkuClaw 是什么？

PkuClaw 是一个 daemon-centered 的 study-agent runtime。它把用户实时消息、定时后台检查、可编辑 runtime 文件、技能目录和渠道通知统一进一套小而清晰的运行模型里。

它当前聚焦 PKU 学习工作流：课程通知、作业与 DDL 汇总、教学网更新检查、后台重要事项提醒，以及后续可扩展的学习任务自动化。

<CardGrid stagger>
  <Card title="两类 Agent run" icon="rocket">
    只有 `realtime` 和 `loop` 两类来源：实时任务直接回复用户，后台任务默认静默，只在重要变化时通知。
  </Card>

  <Card title="文件即 runtime" icon="seti:config">
    `configs/runtime/` 下的 runtime、events、prompts 和 skills 都是可编辑文件，适合热更新和版本管理。
  </Card>

  <Card title="Skill Catalog" icon="open-book">
    Prompt 只注入技能元数据，Agent 在需要时按路径读取具体 skill markdown，避免把所有实现细节塞进上下文。
  </Card>

  <Card title="渠道解耦" icon="puzzle">
    loop 通知通过 channel-neutral outbox 文件队列进入 daemon，再由 channel backend 投递到飞书等平台。
  </Card>
</CardGrid>
