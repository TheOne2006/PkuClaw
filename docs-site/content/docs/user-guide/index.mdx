## 推荐顺序

1. [快速开始](/docs/user-guide/quickstart)：完成最短启动路径。
2. [安装](/docs/user-guide/installation)：补齐环境、provider 和渠道依赖。
3. [配置说明](/docs/user-guide/configuration)：填写本地配置并理解运行时配置。
4. [Realtime Quick Actions](/docs/user-guide/quick-actions)：把常用学习任务做成快捷入口。
5. [FAQ](/docs/user-guide/faq)：排查常见问题。

如果你要修改 PkuClaw 的代码、prompt、runtime loader 或 channel adapter，请转到 [开发者指南](/docs/developer-guide)。
