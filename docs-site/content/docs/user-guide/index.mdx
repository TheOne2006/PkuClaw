这里不展开内部架构、runtime loader 或 channel adapter 的实现细节；如果你要改代码，请看开发者指南。

## 推荐顺序

1. [快速开始](/docs/user-guide/quickstart)：先把仓库、依赖和最小本地配置准备好。
2. [配置总览](/docs/user-guide/configuration)：知道哪些文件需要你填，哪些文件不能提交。
3. [Channel 配置](/docs/user-guide/channel)：如果要用飞书，先让消息能进来、通知能出去。
4. [Agent 配置](/docs/user-guide/agent)：确认 Codex CLI、模型、权限和超时。
5. [Skill 配置与扩展](/docs/user-guide/skills)：把自己的学习任务写成可复用 skill。
6. [PKU 教学网登录](/docs/user-guide/pku-login)：让 PkuClaw 可以读取教学网、作业和课表数据。
7. [Quick Actions](/docs/user-guide/quick-actions)：把常用任务绑定成一键触发入口。
