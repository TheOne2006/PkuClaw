PkuClaw 是一个 daemon-centered study-agent runtime：它把课程/通知类 workflow 拆成清晰的 realtime run、后台 loop、runtime files 和 channel-neutral outbox。

## 核心模型

| 能力 | 说明 |
| --- | --- |
| Realtime | 用户消息和 quick action，直接创建 `source = realtime` run 并流式回复。 |
| Loop | 后台周期任务，默认静默，只在重要变化时通过 outbox 通知。 |
| Runtime Files | `configs/runtime/**` 是可编辑、可 review、可热加载的 runtime surface。 |
| Skill Catalog | Prompt 只注入目录元数据，具体 skill markdown 由 Agent 按需读取。 |
