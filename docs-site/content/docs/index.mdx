PkuClaw 是一个 daemon-centered study-agent runtime：它把课程/通知类 workflow 拆成清晰的 realtime run、后台 loop、runtime files 和 channel-neutral outbox。

文档分成两条完全平行的路径：

<Cards>
  <Card title="使用指南" href="/docs/user-guide" description="安装、配置、启动和日常使用 PkuClaw。" />
  <Card title="开发者指南" href="/docs/developer-guide" description="理解架构、runtime、skill catalog 和贡献流程。" />
</Cards>

## 如何选择

- 如果你要**安装、配置、启动、日常使用**，从 [使用指南](/docs/user-guide) 开始。
- 如果你要**改代码、改 prompt、维护 runtime/skill、提交贡献**，从 [开发者指南](/docs/developer-guide) 开始。
